# ExamenU1
Nombre: Geovanny Romero

>Descripcion
```
Se debe crear un sistema que permita guardar, modificar, y listar las simulaciones de atencion en ventanilla en 
el IESS. La simulacion se trata de lo siguiente: se tiene 3 ventanillas, cada ventanilla atiende personas (no se 
requiere sus datos personales) que van hacer los tramites estos pueden ser: cambio de clave, actualizacion de 
datos, asignacion de afiliados, recuperacion de clave, cada tramite se demora 4 min, 7 min, 15 min, 3 min 
respectivamente. El numero de personas a ser atendidas por cada ventanilla es de forma aleatoria y va minimo 
15 maximo 20, los tramites que van hacer esas personas se debe asignar de forma aleatoria. Al finalizar la 
simulacion se debe tener el total de tiempo de atencion, el nro global de personas atendidos, el nro de personas 
atendidas por cada ventanilla y el total del tiempo de atencion por ventanilla.
```
